# Uber Trip Analysis - Power BI

## Project Overview:
This Power BI dashboard analyzes 103K Uber bookings across:
- Payment Types
- Trip Timing (Day/Night)
- Vehicle Categories
- Booking Value & Distance

## Key Metrics:
- Total Bookings: 103.7K
- Total Booking Value: $1.6M
- Avg Trip Distance: 3 miles

## Tools Used:
- Power BI
- DAX
- Excel (for data prep)
